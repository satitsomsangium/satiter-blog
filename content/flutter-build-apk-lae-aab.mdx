โพสต์นี้เป็น placeholder สำหรับรวบรวมวิธี build แอป Flutter ออกมาเป็นไฟล์ `APK` และ `AAB`

## หัวข้อที่จะใส่ในเนื้อหาจริง

- ความแตกต่างระหว่าง APK และ AAB
- คำสั่ง build ที่ใช้บ่อยใน Flutter
- การตั้งค่า signing สำหรับ Android release
- วิธีตรวจสอบไฟล์ build หลังจาก build เสร็จ

## ตัวอย่างคำสั่ง

```bash
flutter build apk --release
flutter build appbundle --release
```

## หมายเหตุ

เนื้อหาจริงจะเพิ่มภายหลัง โดยจะลงรายละเอียดขั้นตอนใน VS Code และข้อควรระวังตอนปล่อยแอปขึ้น Store
