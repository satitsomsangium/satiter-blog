โพสต์นี้เป็นโน้ตสรุปการใช้งาน `git clone` สำหรับดึงโปรเจกต์จาก GitHub มาไว้ในเครื่องแบบรวดเร็ว

## เป้าหมายของบทความ

- อธิบายคำสั่งพื้นฐานสำหรับ clone repository
- อธิบายความต่างระหว่าง HTTPS และ SSH
- สรุปปัญหาที่เจอบ่อยระหว่าง clone และแนวทางแก้ไข

## ตัวอย่างคำสั่งที่ควรรู้

```bash
git clone https://github.com/owner/repository.git
```

```bash
git clone git@github.com:owner/repository.git
```

## หมายเหตุ

เนื้อหาจริงจะถูกเติมในภายหลัง โดยเฉพาะขั้นตอนการตั้งค่า SSH key และการจัดการสิทธิ์การเข้าถึง repository
